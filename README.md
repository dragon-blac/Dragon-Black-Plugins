# Repositorio gestor de plugins para Dragon-Black

este es la repo de gestor de plugins para Dragon-Black donde puedes compartir tua herramientas 

## distribuir
para integrar tus herramientas en este repo as un fork de este Repositorio
al momento de aser un fork github te dara una copia exacta de este repo 

- 1 clona la copia que github te dara no esta, as un git clone en tu terminal
ejm:
```bash
https://github.com/tu-usuario/Dragon-Black-Plugins

```
- 2 crea una reepo publico para tu plugin 

- 3 publica tu plugin en la repo de tu plugin 

- 4 una vez publicado tu plugin ala repo de tu plugin mueve toda la carpeta donde esta tu plugin ala repo que as clonado de Dragon-Black 
- luego dentro de la carpeta de la repo de Dragon-Black-Plugins en tu terminal ejecuta

```bash
git add .
git push origin main
```

- 6 buelbe ala repo de  Dragon-Black-Plugins que guthub te dio de esta as un pull request y listo
